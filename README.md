📊 Sales Dashboard - Power BI

This project presents an interactive sales dashboard built using Power BI to analyze business performance.

🔹 Key Features:

KPI cards for Sales, Profit, Discount, and Quantity
Monthly sales trend analysis
Sales breakdown by category and sub-category
Geographic visualization of sales distribution
Customer-level insights

🔹 Tools Used:

Power BI
Power Query
DAX

🔹 Insights:

Sales increased in May compared to April
Technology category contributed the highest revenue
Certain customers contributed significantly to total sales
